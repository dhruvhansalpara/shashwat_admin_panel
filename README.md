# Shashwat Admin Panel

This repository contains the admin dashboard for managing banners, destinations, packages, and inquiries for the Shashwat travel site.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
